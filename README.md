# 🤖 AI Chatbot using LangChain & OpenRouter

A simple AI chatbot built with **Python**, **Flask**, **LangChain**, and the **OpenRouter API**. The chatbot provides a clean web interface where users can ask questions and receive AI-generated responses in real time.

## ✨ Features

- 🤖 AI-powered conversations
- ⚡ Fast and responsive chat interface
- 📝 Markdown-formatted responses
- 🔐 Secure API key management using `.env`
- 🌐 Powered by OpenRouter API
- 🎨 Simple and clean UI

## 🛠️ Tech Stack

- Python
- Flask
- LangChain
- OpenRouter API
- HTML
- CSS
- JavaScript

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/your-repository.git
```

### 2. Navigate to the project

```bash
cd your-repository
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Create a `.env` file

```env
OPENROUTER_API_KEY=your_openrouter_api_key
```

### 5. Run the application

```bash
python app.py
```

### 6. Open in your browser

```
http://127.0.0.1:5000
```
